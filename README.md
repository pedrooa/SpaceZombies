# SpaceZombies

Jogo 3D desenvolvido com Unity.

# Jogar

# Objetivo

Pouse no planeta e encontre a espada mágica! Cuidado com os perigos no caminho...

# Como Jogar

## Parte 1
- Virar: setas
- Atirar: tecla alt
- Propulsionar: barra de espaço

## Parte 2 e 3
- Movimente-se com as setas
